Les fonctions du CSS

Règler la police d'écriture

font-size : permet de regler la taille de la police

font-family : permet de changer la police

@font-face : permet de mettre une police personnalisée

font-style : permet de changer le style de la police

font-weight : permet de mettre en gras la police

text-align : permet d'aligner le texte horizontalement

Changer le background

backgound-color : permet de changer la couleur de fond

background-image : permet de mettre une image dans le fond

background-repeat : permet de répeter ou non un element

background-attachement : permet de fixed un element sur le fond

background-size : permet de régler la taille du fond

background-position : permet de régler la position du fond

opacity : permet de régler la transparence

Les boîtes

width : permet de régler la largeur

height : permet de réglér la hauteur

min/max-widht : permet de réglér la larguer min/max

min/max-height : permet de réglér la hauter min/max

margin-top/left/bottom/right : permet de régler la marge en haut/gauche/bas/droite

padding-top/left/bottom/right : permet de régler la marge interne en haut/gauche/bas/droit
